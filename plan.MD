## 5. 기술 스택 (확정)

| 구분 | 기술 | 비고 |
|------|------|------|
| 프론트엔드 | React (Vite) | SPA, JSX, hooks |
| 데이터베이스 | Supabase (PostgreSQL) | 게시글 저장/조회/삭제 |
| 배포 | GitHub Pages | gh-pages 패키지로 자동 빌드·배포 |
| 버전 관리 | Git + GitHub | main 브랜치 커밋 관리 |

### 아키텍처 구조 (v2 현재)

```
[사용자 브라우저]
       │
       ▼
[React 앱 - GitHub Pages]
       │  REST API (Supabase JS SDK)
       ▼
[Supabase - PostgreSQL DB]
  posts 테이블
  (id, title, content, category, tags,
   type, image_data, video_url, date)
```

> 별도 백엔드 서버 없이 Supabase를 BaaS로 활용.
> 추후 로그인 기능 추가 시 Supabase Auth 연동 예정.
