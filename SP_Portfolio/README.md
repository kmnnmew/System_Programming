# 심하윤 포트폴리오

> React + Supabase 기반 개인 포트폴리오 & 학습 게시판

🌐 **배포 URL:** https://kmnnmew.github.io/System_Programming/

---

## 기능

### v1 — 포트폴리오
- 히어로 / About Me / 기술 스택 / 프로젝트 / 수상 경력 / 연락처

### v2 — Study Board 추가 (`/#v2` 진입)
- 학습 내용 기록용 게시판 (텍스트 · 이미지 · 동영상)
- 카테고리 필터 (React / JavaScript / Java / Python / DB)
- Supabase DB 연동 — 글쓰기 · 조회 · 삭제
- YouTube URL 썸네일 자동 미리보기

---

## 기술 스택

| 구분 | 기술 |
|------|------|
| 프론트엔드 | React (Vite) |
| 데이터베이스 | Supabase (PostgreSQL) |
| 배포 | GitHub Pages |

## 실행 방법

```bash
# 패키지 설치
npm install

# 개발 서버
npm run dev

# GitHub Pages 배포
npm run deploy
```
